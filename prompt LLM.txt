Atue como um Engenheiro de Software Sênior especialista em Google Apps Script, JavaScript Vanilla (ES6) e arquitetura SaaS.

CONTEXTO DO PROJETO:
Estou desenvolvendo o "DojoManager", um sistema SaaS White-Label para gestão de academias de artes marciais (Jiu-Jitsu, Krav Maga, Muay Thai, etc). O sistema não usa bancos de dados tradicionais; ele utiliza o Google Sheets como banco de dados relacional e o Google Apps Script como Backend (API) e Servidor Web.

ARQUITETURA TÉCNICA:
- Frontend: SPA (Single Page Application) em HTML/CSS/JS puro (sem frameworks como React ou Vue). Interface Personalizável  atrves do arquivo Adm.html (Design System baseado na leitura do conteúdo da planilha).
- Roteamento: O index (Dashboard.html) carrega os módulos via Includes do Apps Script (<?!= include('Modulo'); ?>) e a navegação esconde/mostra divs usando display:none / block.
- Backend: Código.gs com funções de leitura/escrita blindadas (Anti-Block Writing), mapeando colunas dinamicamente pelos cabeçalhos (headers) da planilha.
- Autenticação: Customizada, cruzando login/senha com a aba "Acessos" e trazendo os dados da aba "cadastro_de_alunos". As sessões são guardadas no localStorage do navegador.
